What is it?

Catquiz was catsopinion.com blog's content quiz. It was a funny addition to static written blog content and was actually quite informative. The quiz can be played in Finnish or English. If I can recall correctly, this was the first typescript project I did. Quiz is filled with animation and is responsive. I went for a detective theme and the small magnifying glass at the bottom was really fun to make. What I regret the most is the fact that I'm not a very good writer and questions contain writing errors and some brain farts.

Goals and lifespan

I did not have any specific goals for this in mind. I just wanted to make a quiz about cats, it seemed like a fun project to make. At least I did get in touch with typescript and I liked it. The quiz is still up and available to try.

Image from the gameplay

Tech used

Catquiz is a statically hosted single page application. It is written in react. Data is stored in the Firebase realtime database and the database was filled with python script that can be found here. The script was written fast and needed only once, so it might not be pretty. What I did was, I wrote questions in Excel type charts. Exported those to csv and then added the data to the database. Data was gathered all around the internet manually.

Catquiz site can be found at here. Catquiz code can be found at Github repo. Python script can be found at Github repo.
