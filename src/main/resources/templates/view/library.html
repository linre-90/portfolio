Virtual library

27.12.2022

This was a project that was ordered from us and the goal was to create some sort of game that shows kids things about library.

The main menu of the game. The pig was a mascot that was made from paper mass for the library. So we digitalized it.

What is it?

This was our intern section project. We were asked to make a game for the local library. The only restriction was to include pigs and that the facts were correct. We created a virtual detailed model from the library and put the player in as a robot. Pig gives players missions that make the player familiar with the library and its layout.

Picking a player also customizes book suggestions given by a pig, so a character playing as a princess robot is more likely to receive books about princesses. Pig also tells really corny jokes like: "What is the biggest rice in the world?" - "Paris"... Well, jokes work better in my native language.Controls were made simple and bright moving items were added to make the library more fun place.

Pig was able to fly around the library and act as a host for the player.

Sections were color coded and illustrated with images.

Our team name was vihriä omena production, and we placed "advertisement" items around library.

The result

The game turned out great, and we managed to put a lot of activities in it. We also received a diploma about the game.

The game featured working lending machines and other mini-games inside computers that were placed around the library.

We optimized game heavily to run on low-end hardware.

Players move around as robot. Complete assignments like book rally or visit sections.
